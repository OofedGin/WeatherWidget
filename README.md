# WeatherApp

Current weather and weather forecast app based on PyQT
### Guide
- pip install these:
  - PyQt6
  - PyQt6_WebEngine 
  - requests
  - folium
- Get your API tokens here:
  - [weatherapi](https://www.weatherapi.com/)
  - [ipgeolocation](https://app.ipgeolocation.io)
- insert them into config fields
- launch main.py
